# Claude Code 101

Interactive onboarding skill for Claude Code beginners.

## Install

터미널에서 한 줄 실행:

```bash
curl -sL https://raw.githubusercontent.com/ian-marq/claude-101/main/plugins/cc101/skills/onboarding/SKILL.md --create-dirs -o ~/.claude/skills/onboarding/SKILL.md
```

## Usage

Claude Code에서:

```bash
/onboarding
```

Claude will walk you through the basics step by step:
1. Claude / Cowork / Code 차이점
2. 파일 읽기 체험
3. 실행 방법 (터미널 / VS Code / Desktop App)
4. 유용한 명령어 (`/cost`, `/model`, `/compact`)
5. Plan 모드 체험
6. MCP 연동 소개
7. 기억 체계 (CLAUDE.md, Rules, Skills)
8. `/clear` & `/init` 체험
