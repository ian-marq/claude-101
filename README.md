# Claude Code 101

Interactive onboarding plugin for Claude Code beginners.

## Install

```bash
# In Claude Code:
/plugin marketplace add ian-marq/claude-101
/plugin install claude-101@ian-marq-claude-101
```

## Usage

```bash
/claude-101:onboarding
```

Claude will walk you through the basics step by step:
1. File reading and navigation
2. Key commands (`/cost`, `/model`, `/compact`)
3. Plan Mode vs Normal Mode
4. Free exploration

## About

Created for MarqVision's internal Claude Code 101 session.
